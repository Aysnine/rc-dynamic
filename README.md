## Getting Started

``` bash
# install dependencies
yarn install

# development mode
yarn serve

# build production
yarn build

# check code styles
yarn lint:script

# check types
yarn type

# show storybook
yarn storybook

# show storybook
yarn storybook

# build storybook
yarn build-storybook
```
